# Как установить tmux
- Скачать и установить [tmux](https://github.com/tmux/tmux/wiki/Installing)
- В терминале MacOS поставить галочку в настройке "Use Option as Meta key"
- Для активации плагинов нужно установить [tmp](https://github.com/tmux-plugins/tpm)
- Для применения конфига tmux нужно прописать `tmux source-file ~/.tmux.conf`
- Для активации плагинов внутри tmux нажать `prefix` + `I`
## Шрифт и иконки
- Установить [JetBrains Mono Nerd Font](https://www.nerdfonts.com/font-downloads)
- Для MacOS положить `*.ttf` файлы в `~/Library/Fonts`
- В терминале выбрать `JetBrains Mono` шрифт
